New! Design APIs Directly in Postman


We're excited to announce Postman's new feature for creating APIs!

Manage every aspect of your API with the new API tab and versioning. The new update is packed with new features that make API design and development quicker and easier.


Here's what you'll get when you update to Postman v7.1:

  • The API tab creates an organized workflow that is centered around your API so you can design and develop from start to finish in Postman.
  • Extended schema support allows you to write, edit, or import your API schema to define your API.
  • API versioning allows you to create and maintain multiple versions of an API simultaneously.
  • Version tagging allows you to tag and link API elements like Postman Collections, mock servers, and variables to any version of your API.
